# Source Checklist

Use this checklist before making public claims in README, grant submission or post.

## Verify against official sources

- Arc official docs.
- Arc official community/events/resources.
- Circle Developers docs.
- Circle Agent Stack docs.
- Circle Gateway/Nanopayments docs.
- Circle Grants page.
- Questbook/Circle grants page if used.
- x402 docs if integration is claimed.

## Claims that require verification

- Current grant eligibility.
- Current deadlines.
- Current reward amounts.
- Current Arc House points/rules.
- Current API parameters.
- Current Circle product names.
- Supported networks/chains.
- Production availability.
- Pricing/limits.

## Safe wording

Use:

- "designed for";
- "compatible architecture";
- "preflight layer";
- "proof-of-concept";
- "MVP";
- "intended integration path".

Avoid unless verified and implemented:

- "integrated with";
- "official";
- "approved";
- "production-ready";
- "compliance-grade";
- "guaranteed".
