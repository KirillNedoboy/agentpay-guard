# package.intent.md

This starter pack intentionally does not include application code.

When implementing, use a compact TypeScript fullstack setup.

Recommended stack:

- pnpm;
- TypeScript;
- Next.js App Router or another compact fullstack TS framework;
- deterministic test runner;
- ESLint;
- typecheck command.

Expected commands after implementation:

```bash
pnpm install
pnpm dev
pnpm test
pnpm lint
pnpm typecheck
```
