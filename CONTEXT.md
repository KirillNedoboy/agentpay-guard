# CONTEXT.md

## Arc/Circle narrative

AgentPay Guard is built for the emerging agentic/stablecoin payment environment where agents and machines can pay for API access, compute, data, telemetry and services.

The product is relevant to:

- Arc agentic economy narrative;
- Circle Agent Stack;
- Circle Gateway / Nanopayments;
- x402 paid API flows;
- machine-to-machine payments;
- builder/grant proof requirements.

## Product thesis

Payment rails enable autonomous spending.

AgentPay Guard provides a preflight decision layer before autonomous spending.

## Avoid weak narratives

Do not position this as:

- another wallet;
- a generic payment app;
- a trading bot;
- a compliance product;
- a fraud guarantee;
- a smart contract protocol.

## Strong narrative

Position it as:

> A programmable payment firewall and audit trail for autonomous AI-agent payments.

## Important wording

Use:

- preflight;
- policy;
- audit;
- decision engine;
- payment intent;
- x402-compatible flow;
- Gateway-compatible flow;
- stablecoin commerce;
- agentic payments.

Avoid:

- AML;
- guaranteed safety;
- official module;
- certified compliance;
- production custody.
